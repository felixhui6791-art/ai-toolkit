# Windows 版 · 第一步：打地基

目标：先确认 **Python + Ollama + 大模型** 这条基础链路在你的 Windows 上能跑通。
（这一步通了，后面才往上加语音、截图、控制面板等。）

## 1. 装 Python
- 去 https://www.python.org/downloads/ 下载安装。
- ⚠️ 安装第一屏**勾选「Add Python to PATH」**，再点 Install。

## 2. 装 Ollama（本地大模型的发动机）
- 去 https://ollama.com/download 下载 Windows 版，安装并打开。
- 打开「命令提示符」(开始菜单搜 cmd)，输入并回车下载一个模型：
  - 内存 8GB：`ollama pull qwen3:4b`
  - 内存 16GB：`ollama pull qwen3:8b`
  - 内存 24GB 以上：`ollama pull qwen3:14b`

## 3. 运行检查
- 双击本文件夹里的 **`检查环境.bat`**。
- 如果看到 **「基础链路通了！」** 就成功了 —— 把窗口截图发我，我们进下一步。
- 如果报错，也把截图发我。
